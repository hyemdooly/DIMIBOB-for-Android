# DIMIBOB-for-Android
KDMHS Meals Widget Project - Android Platform

한국디지털미디어고등학교 급식 위젯 앱 서비스입니다.

플레이 스토어에 업로드하여 학생들에게 배포했지만 급식 API가 닫혀 지금은 배포 중이지 않습니다.

사용 언어: Java
사용 기술: Realm
